This repository hosts the generated API documentation for the pyxora game engine.<br>
Only the latest version of the documentation is stored here for space reasons.

## Viewing Docs Locally
You can use the pyxora CLI to view the documentation without visiting the website:
```bash
# Open docs online in your browser
pyxora docs online

# Open docs locally (served via localhost)
pyxora docs local

# Build the docs into static HTML
pyxora docs build
```

## Contributing

This repository is automatically generated from the main pyxora repo using pdoc.<br>
Any changes to the documentation must be made in the docstrings of the main pyxora repository.
